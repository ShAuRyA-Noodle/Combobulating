# Phase 3 Stage Kit — Pack List

Galaxy Brain — Aura — Samsung EnnovateX 2026.

Stage day at Taj Yeshwanthpur, Bangalore. The kit must support the
5-minute live demo per `plan.md` §28 and the 90-second backup video
per `plan.md` §28 ("Backup").

Pack the night before travel. Verify everything works on Mac the
morning of departure.

---

## 1. Devices (must-have)

- [ ] iPhone — primary demo device. Aura installed via TestFlight, all
  permissions pre-granted. Lock screen passcode disabled or known.
- [ ] Apple Watch — paired to iPhone. HRV data simulated for the
  demo (do not rely on stage-day biometric since the user is in
  fight-or-flight on stage and HRV will tank).
- [ ] AirPods (Pro 2 if available) — for the on-stage earbud whisper
  if the venue audio chain supports Bluetooth output. If venue is
  speaker-only, route via Mac and skip AirPods.
- [ ] MacBook — Phase 1 deck open in Keynote, Phase 3 deck in
  Keynote, demo video pre-loaded, AVD running, browser open to
  GitHub repo.
- [ ] Mini-projector adapter — HDMI **and** USB-C / Thunderbolt
  varieties. Pack both. Venue might have either.
- [ ] Battery banks × 2 — 20,000 mAh each, fully charged. iPhone +
  Watch + Mac drain hard during a 5-minute demo with screen
  mirroring.
- [ ] Lavalier mic — wired 3.5 mm + USB-C. Wireless lav is too
  finicky on a Samsung-engineer-judged stage.
- [ ] Laser pointers × 2 — one for each presenter. Fresh batteries,
  test at home.

---

## 2. Cables (no compromise)

- [ ] USB-C to USB-C (Mac → external drive, Watch → AirPods case)
- [ ] USB-C to Lightning (Mac → iPhone backup demo)
- [ ] HDMI cable — 3 m, full-size (venue might give a stub)
- [ ] USB-C to HDMI adapter
- [ ] Lightning to HDMI adapter (iPhone direct projection backup)
- [ ] 3.5 mm aux cable (venue audio backup)
- [ ] USB-A to USB-C dongle (venue laptop port backup)

Pack in a single Ziploc with a label "STAGE CABLES — DO NOT REMOVE".

---

## 3. Printed leave-behinds (50 copies)

Per `plan.md` §26 (₹1,500 print budget) and `aura/design/leave_behind/a4_layout.md`.

- [ ] 50 × A4 colour leave-behind sheets — front: hero + one-liner +
  QR to GitHub; back: KPI evidence block + judges memo single
  sentences.
- [ ] 20 × business cards (optional; ₹0 if printed at home).

Carry in a stiff folder so corners do not bend.

## 4. Documents

- [ ] EnnovateX badge confirmation printout (digital + paper)
- [ ] Train ticket PNR printouts (round trip)
- [ ] Hostel booking printout
- [ ] Insurance proof if Thapar provides (not mandatory)

## 5. Demo redundancy

Per `plan.md` §33 Failure Mode 10 ("Demo failure on stage"):

- [ ] 90-second pre-recorded backup video on Mac desktop in
  `~/Demos/aura_backup_90s.mp4`
- [ ] Same video also on iPhone Photos
- [ ] Same video also on a USB-A flash drive (8 GB minimum)
- [ ] Phase 3 deck PDF on the same flash drive
- [ ] All passwords (TestFlight, GitHub, AVD admin) in a paper
  envelope sealed and only opened if the digital password manager
  fails. Burn after the event.

## 7. Stage-day morning ritual

| Time | Action |
|---|---|
| T-3 h | Wake; full breakfast at hostel |
| T-2 h | Charge everything to 100% on hostel desk; pack |
| T-90 min | Uber to venue; arrive early |
| T-60 min | Find on-site tech rehearsal slot if offered; mirror Mac to projector; run Aura demo end-to-end |
| T-30 min | Mic check; lavalier pinned 6 inches below collar; voice level confirmed |
| T-15 min | Open laptop on stage; deck pre-loaded; demo video buffered; phone Wi-Fi off, mobile data on |
| T-5 min | Both presenters in position; one final breath cycle; locked opening line in Shaurya's head |
| T-0 | Walk on. Open with the locked Phase 3 line per `phase3/README.md`. |
